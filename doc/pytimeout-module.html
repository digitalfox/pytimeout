Module pytimeout

source code

pytimeout module. Provide both a command line interface and an API (via the "run" function) to allow process launching with a define time to live before being killed.

run(target, timeout=TIMEOUT, retry=RETRY, preserveChildren=False, verbose=False)

source code 
Run a process but kill it after timeout
Parameters:
  • target - the command to launch (str)
  • timeout - timeout in seconds before start killing process (int)
  • retry - Number of time we will try to kill the process with SIGTERM and SIGKILL (int)
  • preserveChildren - Do we need to also kill process children ? Default is True (bool)
  • verbose - Print what happened on standard output (bool)
Returns:
True if everything is ok. None or False else.
